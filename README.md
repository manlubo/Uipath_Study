# Study UiPath Project Folder Guide

이 문서는 UiPath 프로젝트에서 사용하는 주요 폴더의 역할을 정리한 안내서입니다.

| 폴더 | 역할 | 주로 넣는 파일 예시 |
|---|---|---|
| `Config/` | 프로젝트 설정값을 보관합니다. 환경별 값이나 실행 옵션을 관리할 때 사용합니다. | `Config.xlsx`, `Settings.json`, 환경별 설정 파일 |
| `Data/` | 자동화 실행 중 사용하는 업무 데이터를 모아두는 상위 폴더입니다. | 입력/출력/임시 데이터 하위 폴더 |
| `Data/Input/` | 자동화가 읽어야 하는 원본 입력 데이터를 보관합니다. | 처리 대상 Excel, CSV, PDF, 텍스트 파일 |
| `Data/Output/` | 자동화 실행 결과물을 저장합니다. | 결과 Excel, 생성된 리포트, 다운로드 파일 |
| `Data/Temp/` | 실행 중 잠시 필요한 임시 파일을 저장합니다. 실행 후 삭제 가능한 파일을 둡니다. | 중간 변환 파일, 임시 다운로드 파일 |
| `Docs/` | 프로젝트 설명, 업무 절차, 설계 문서 등 사람이 읽는 문서를 보관합니다. | PDD, SDD, 운영 매뉴얼, 테스트 시나리오 |
| `Exceptions/` | 예외 처리와 관련된 참고 파일이나 예외 로그 샘플을 보관합니다. | 예외 유형 목록, 재처리 기준 문서 |
| `Framework/` | 공통 처리 로직이나 프레임워크성 워크플로를 보관합니다. | 공통 초기화, 공통 예외 처리, 공통 로깅 워크플로 |
| `Logs/` | 실행 로그나 디버깅용 로그 파일을 보관합니다. | 실행 로그, 에러 로그, 디버그 출력 |
| `Resources/` | 자동화에서 참조하는 정적 리소스를 보관합니다. | 템플릿 파일, 이미지, 메일 본문 템플릿 |
| `Tests/` | 테스트용 워크플로, 테스트 데이터, 검증 자료를 보관합니다. | 테스트 케이스, 테스트 입력 파일, 예상 결과 파일 |
| `Workflows/` | 메인 워크플로 외의 세부 워크플로를 모아두는 상위 폴더입니다. | 업무 단계별 `.xaml` 파일 |
| `Workflows/Init/` | 실행 시작 시 필요한 초기화 워크플로를 보관합니다. | 설정 로드, 로그인, 사전 점검 워크플로 |
| `Workflows/Process/` | 실제 업무 처리를 수행하는 핵심 워크플로를 보관합니다. | 데이터 처리, 화면 입력, 파일 생성 워크플로 |
| `Workflows/End/` | 실행 종료 시 필요한 마무리 워크플로를 보관합니다. | 로그아웃, 결과 저장, 알림 발송, 정리 작업 |
| `Workflows/Components/` | 여러 워크플로에서 재사용할 수 있는 작은 단위의 컴포넌트를 보관합니다. | 공통 클릭/입력 처리, 파일 검증, 문자열 정리 워크플로 |

## 기본 사용 기준

| 기준 | 설명 |
|---|---|
| 입력 파일 | `Data/Input/`에 저장합니다. |
| 결과 파일 | `Data/Output/`에 저장합니다. |
| 실행 중 임시 파일 | `Data/Temp/`에 저장하고, 가능하면 실행 종료 시 삭제합니다. |
| 업무 로직 | `Workflows/Process/`에 단계별로 분리합니다. |
| 반복 사용 로직 | `Workflows/Components/` 또는 `Framework/`에 분리합니다. |
| 설정값 | 코드나 워크플로에 직접 쓰지 말고 `Config/`에서 관리합니다. |
| 문서 | 개발/운영자가 참고할 내용은 `Docs/`에 보관합니다. |

## UiPath 관리 폴더 주의사항

| 폴더 | 주의사항 |
|---|---|
| `.local/` | UiPath CLI와 Studio가 사용하는 로컬 캐시입니다. 직접 수정하지 않습니다. |
| `.objects/` | Object Repository와 UI selector 정보가 관리되는 폴더입니다. Studio 또는 전용 도구로 관리합니다. |
| `.project/` | 프로젝트 메타데이터가 저장됩니다. 직접 수정하지 않습니다. |
| `.settings/` | Studio 설정 정보가 저장됩니다. 필요한 경우 Studio에서 변경합니다. |
